# The Corridor

The environment is based on sci-fi elements in the near future, created over a few weeks and rendered in Arnold Maya. While working on this project I've also used cloud rendering service. Post-process includes adding lights and color adjustment (chroma aberration, contrast, blur, particles, etc).
